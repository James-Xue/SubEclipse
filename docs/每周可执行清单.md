# 字幕遮挡器每周可执行清单（首版 8 周）

版本：v1.0
日期：2026-03-11

## 第 1 周：工程骨架与构建打通
- 建立目录：`src/app`、`src/platform/x11`、`src/vision`、`src/render`、`src/metrics`、`include/subeclipse`、`tests`。
- 创建顶层 `CMakeLists.txt` 与 `cmake/Dependencies.cmake`。
- 打通最小可运行程序：`main.cpp` 启动空窗口并正常退出。
- 接入日志与配置读取（可先 JSON）。
- 验收：`cmake -S . -B build && cmake --build build -j` 成功。

## 第 2 周：ROI 交互与 Overlay 基础
- 实现 ROI 新建/移动/缩放。
- 实现透明 Overlay 窗口（先不接检测）。
- 实现点击穿透开关（编辑 ROI 时关闭，运行时开启）。
- 接入热键：开始/暂停/重选 ROI/退出。
- 验收：能手动框选 ROI，Overlay 可见且不挡鼠标操作。

## 第 3 周：X11 抓屏链路
- 实现 `IScreenCapture` 与 `X11Capture`。
- 优先 `XShmGetImage`，失败回退 `XGetImage`。
- 实现 ROI 局部抓屏与时间戳标注。
- 增加抓屏中断重连与错误提示。
- 验收：可稳定输出 ROI 帧流，20 FPS 目标达成。

## 第 4 周：检测与基础遮挡
- 接入 `ITextDetector`（先占位/简版检测器也可）。
- 输出 `TextDetection` 并在 Overlay 上画黑框遮挡。
- 建立 `capture -> vision -> render` 队列管线。
- 增加核心参数配置项（阈值、FPS、样式）。
- 验收：ROI 内文本大体可被遮挡，非文本保持透明。

## 第 5 周：跟踪稳定与抖动抑制
- 实现 `TrackManager`（IoU 关联 + 生命周期）。
- 增加迟滞阈值：`score_enter` / `score_keep`。
- 增加 EMA 平滑（位置与尺寸）。
- 加入最小生命周期与最大丢失帧策略。
- 验收：闪烁率和抖动明显下降，稳定字幕效果可用。

## 第 6 周：样式、容错与指标
- 完成三种遮挡样式：黑框/模糊/像素化。
- 完成模型失败、抓屏中断、配置损坏容错。
- 增加指标采集：`fps_current`、`latency_p50/p95`、掉帧率。
- 增加状态机与错误态提示（Degraded/Error）。
- 验收：异常可恢复；性能指标可持续输出。

## 第 7 周：Wayland 受限模式
- 实现 Portal 授权与 PipeWire 帧接入基础流程。
- 完成 L1/L2/L3 降级判定与 UI 提示。
- 保证“受限模式可运行”，不强行承诺全局置顶。
- 验收：Wayland 下可授权采集并完成局部遮挡预览。

## 第 8 周：联调、回归与发布准备
- 完成 4 小时长稳测试与多 DPI 回归。
- 进行性能优化与参数收敛。
- 生成默认配置、运行说明、故障排查文档。
- 完成 MVP 交付打包（可先 tar.gz）。
- 验收：满足需求文档中的 AC-01 ~ AC-12 核心条目。

## 每周固定节奏（建议）
- 周一：拆任务、冻结本周验收口径。
- 周二~周四：实现 + 自测 + 指标记录。
- 周五：联调回归、更新文档与风险清单。
